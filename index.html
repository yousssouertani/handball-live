⚙️ CONFIGURATION

Pour que les scores soient synchronisés en temps réel sur tous les appareils, l'appli utilise JSONBin.io (gratuit). Suis ces étapes une seule fois :

  1. Va sur jsonbin.io → Crée un compte gratuit
  2. Clique CREATE BIN → colle {} → Save
  3. Copie le BIN ID (ex: 6660abc123...)
  4. Va dans API Keys → copie ta Master Key
❌ Erreur — vérifie le BIN ID et l'API Key
